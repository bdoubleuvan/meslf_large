## Description

Code from (https://github.com/bdoubleuvan/meslf) modified for large energy networks based on the work of A.S. Markensteijn (https://doi.org/10.4233/uuid:044c083a-4eb1-4999-b789-db7150c4c7df). 
This version includes large coupled electricity and gas networks. Speed-up in creating Jacobian and non-linear system of equations. 
Options for preconditioners with Krylov solvers from scipy and PETSc.
